# esp-idf-m5stick
Sample code for driving M5Stick OLED display using ESP-IDF's SPI driver.   

```
git clone https://github.com/nopnop2002/esp-idf-m5stick
cd esp-idf-m5stick/
make menuconfig
make flash
```
